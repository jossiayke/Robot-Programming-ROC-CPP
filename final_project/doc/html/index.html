Final Project: US&R Simplified Documentation

This project is inspired by the challenge of autonomous robotics for Urban Search and Rescue (US&R). In US&R after a disaster occurs, a robot is used too explore an unknown environment, such as a partially collapsed building, and locates trapped or unconscious human victims of the disaster.

  • In this project, we will be simulating this scenario using 2 Turtlebot3 Waffale model robots.
  • The first robot is called "expolrer" which has the responsibility of discovering the map and locate the AruCo markers (victims). Then send the the location information to the second robot "follower". The explorer will head back to base location.
  • follower robot is going to follow the path generated based on the information given by "explorer". The follower is responsible for rescuing the victims in the order given by "explorer". After rescuing all the victims, the "follower" will head back to base location.
  • Per this project, there will be 4 AruCo markers distributed across the map, each one will have a unique ID and fiducial square barcode.
